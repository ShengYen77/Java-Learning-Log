# Client/Server模型 基礎概念
| 元件          | 說明                         |
| ----------- | -------------------------- |
| Client（客戶端） | 向伺服器發送請求的使用者端（例如瀏覽器、手機App） |
| Server（伺服器） | 接收請求並處理，返回資料或操作結果給 Client  |
