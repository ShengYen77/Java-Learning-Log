## ⭐HTTP 方法的屬性 (Attribute)

用來告訴 ASP.NET Core 這個 Action 要接受哪一種 HTTP 方法請求

| Attribute      | 對應 HTTP 方法 | 說明       |
| -------------- | ---------- | -------- |
| `[HttpGet]`    | GET        | 查詢資料     |
| `[HttpPost]`   | POST       | 新增資料     |
| `[HttpPut]`    | PUT        | 全量更新     |
| `[HttpDelete]` | DELETE     | 刪除資料     |
| `[HttpPatch]`  | PATCH      | 局部更新     |
